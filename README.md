# 以下记录本人患甲状腺癌的一些情况（持续更新）

## 2022年8月20日

我在贵港市东晖医院体检，首次查出甲状腺右叶囊性结节（TI-RADS：2级），桥本氏甲状腺炎。后来咨询医生得知，桥本氏甲状腺炎治疗不了，这个是自身免疫疾病，是自身免疫系统攻击甲状腺导致的。医生叫我定期观察即可。

## 2023年10月14日

在贵港市人民医院体检，查出右叶中段实性结节（TI-RADS：3类），大小1.2*0.7cm，边界清晰，低回声，内未见钙化，另：桥本氏甲状腺炎。与2022年体检时相比，说明这个结节增长了。挂号咨询了医生，依然叫我定期观察即可。

## 2024年8月8日

2024年8月8日本人到贵港市人民医院第二门诊体检，B超检查发现甲状腺结节至少3个：甲状腺右叶局灶性病变(一),TI-RADS 4a类；甲状腺左叶局灶性病变(一),TI-RADS 3类；甲状腺左叶局灶性病变(二),TI-RADS 2类。突然一下子检查这么多结节，而且还是TI-RADS 4a类，我也不知道是甲状腺结节一年不到的时间里突然长得这么快，还是之前就已经左右叶两边都长有了。第二天挂号了医生，建议我做活检。8月9日在贵港市人民医院做细针穿刺活检，病理分析结果：双侧甲状腺乳头状癌。

## 得知患癌一时懵逼住了，几天睡不好觉，也想不通为啥我就患癌了

得到检查结果的时候，感觉一下子晴天霹雳，有点接受不了现实，想不通，我每年都体检，为啥突然患癌了？而且我听说甲状腺癌是个懒癌，平时都是以年为单位的增长，为啥我的结节长得这么快？于是开始不断反思，可能是我近年来生活作息不规律，经常熬夜的原因；也可能是暴饮暴食，身体严重长胖的原因；朋友还说，是因为我长期献血，导致身体抵抗力下降……具体是什么原因，我不知道。问了医生，医生说献血一般不会导致抵抗力差，更不会导致患甲状腺癌，患甲状腺癌跟很多因素有关，比如遗传、生活习惯等等。遗传因素我感觉不太可能，家里的亲属没有患这个癌的，虽然我爸去年也查出甲状腺结节4a类，但穿刺检查是良性。而且身边的朋友和同事也很多患有甲状腺结节，感觉这个疾病更多的跟南方的气候和饮食习惯有关吧。但不管什么原因，我都得坦然勇敢面对这个现实。

## 2024年8月28日做了甲状腺全切手术

2024年8月21日，我到广西医科大学附属肿瘤医院（广西壮族自治区肿瘤医院）申请住院治疗，诊断：双侧甲状腺恶性肿瘤。住院做了各种检查，一周左右后，于8月28日做甲状腺全切手术。因为有同事在这家医院做过这个手术，有介绍到肿瘤医院耳鼻咽喉头颈外科副主任汪多平（主任医师，博士）比较厉害。于是我要求我的手术主刀医生是汪多平主任医师，虽然住院安排给我的主管医生是罗永标（主治医师），但听说就是汪多平给我做的手术。手术后，咽喉部位肿痛，吞咽困难，喝开水都被呛到。最明显的症状是声音嘶哑，至今（9月12日）没有恢复正常，医生说需要几个月恢复。术后，需要终身服用优甲乐补充甲状腺素，每天早上起床服用2片，1个小时后吃东西，4个小时内不能吃蛋白类食物，如大豆、牛奶、豆制品、蛋等等。还给我开了一个月的钙片（补钙）和甲钴胺片（修复神经）。一个月后再复查。

手术前，罗永标医生给我和家属做了术前谈话，主要内容是说手术有风险，需要家属和本人签字同意手术，手术可能会损伤喉返神经（说话嘶哑、喝水呛水等后遗症），可能会损伤甲状腺旁腺（缺钙等后遗症），如果是损伤了神经，一般需要3-6个月左右恢复，如果永久性损伤则终生无法恢复。住院期间，微信咨询了我的主管医生，但多数不回复，倒是手术后第三天查房时开始不停地催我赶快办理出院，还说换别人早就出院了，我也不知道为啥，我也没欠费，而且刚搬来新院区病床也不紧张。于是我厚着脸皮多住院了几天，因为我咽喉还肿痛，并且胸口的手术伤口还在隐痛和渗血水（每天从引流管流出大概30几毫升血水），手术后一周左右我才办理出院。可能催我出院也有他的原因吧，表示理解。此外，住院期间，所有医生、护士都对我很好。住院期间，汪多平主任来查房时曾跟我说，我的手术还是做得很成功的，叫我安心休养，缺钙和声音嘶哑的问题一般一个月左右就可以恢复正常。

## 手术前和手术后做的一些检查和对比

手术前后做了血液、甲状腺功能、甲状旁腺等方面的检查，对比了一下，甲状腺旁腺功能好像影响不大，但抽血检查的时间是2024年8月29日早上8点左右，而我的手术是大概8月28日下午17时左右做完手术出来，不知道准确性如何，希望真的没有影响甲状旁腺功能吧。另外，手术前（8月22日）检查血小板计数是212，手术后（8月29日早上8点）检查血小板计数是246，反而增加了，不知道咋回事。
术前医生说，我的甲功方面，促甲状腺激素测定TSH值5.36有一点偏高，也属于甲功不太正常。此外，手术后，甲状旁腺两项检查的降钙素（CT）是0.74，相比手术前的值9.84，相差了很多，难道是手术损害了甲状旁腺功能，担心会不会终身缺钙。住院时我也问过汪多平主任医师，他说不会，不用一辈子补钙的，钙片先吃一个月，后面就正常了。专业医师都这么说了，那后续再检查看看了，希望好起来。缺钙的表现是：四肢麻木，手脚抽筋等。
目前按医生的要求是，每天吃2片优甲乐（补充甲状腺激素的），甲钴胺片每天3次（一次一片，修复受损神经的），钙片先吃几天，后续如果缺钙再持续吃。

手术后等了一个多月，去复诊时才打听到病理报告出来了。奇怪，期间也没人通知我可以看术后的病理报告了。我打印了出来，从报告上来看，右侧是确诊了甲状腺乳头状癌。但是左侧就不能明确是不是甲状腺癌，医生说看起来像是，但是从病理上看没有100%确认。报告上说，可以到病理科继续做免疫组化，进一步确认是否癌症及性质。我去询问了主任医生汪多平博士，他建议不用再去做免疫组化了，反正现在都已经全部切了，做不做都没啥意义了。
但是退一步讲，如果左侧的不是甲状腺癌，说不定原来是不用切的呢？要知道，哪怕还保留有一边的甲状腺，也还可以正常分泌甲状腺功能，后遗症和身体状况都大不一样。我现在做了全切，连左边的也切了，感觉如果左侧不是癌症的话，就有点过度治疗了。哎，没办法，现在已经切了，不想了，只能这样了。

## 本次甲状腺癌医疗的全部花费和保险报销情况

全部算下来，本人这次住院期间花费27425.91元，个人自付10608.26元，不含伙食费。住院半个月，伙食费大概是600元左右。至于后续的费用，1个月后复查，后面需每三个月定期到医院复查等费用另计。加上之前在贵港市人民医院的开支，做B超、活检化验等等费用，总体算下来，大概3万左右吧。不过好在我买了保险，除了单位的职工医保报销了16817.65元，还有工会的大病互助医保赔了851.79元，众惠财产相互保险社赔了1426.09元，还有我自己购买的支付宝好医保也赔了很多。实际上相当于这次治疗不用花什么钱，所以建议大家能买保险就买，以备不时之需。

## 谈谈甲状腺癌全切手术后对生活的影响（持续更新）

下面通过本人亲身经历谈谈甲状腺癌全切手术后对生活造成的影响，这也是大部分病友都关心的问题。当时广西肿瘤医院的汪多平主任和罗永标医师、贵港市人民医院的覃俊仕主任医师都跟我说，甲状腺全切后理论上不会影响寿命，以后该怎么生活就怎么生活，该活100岁就是100岁。我问那一辈子吃优甲乐，会不会有副作用？罗医生说，副作用就是长期服用可能影响到心脑血管方面和肠胃功能方面，另外优甲乐是激素，吃多吃少都会有问题，所以需要定期复查以调整优甲乐的服用量。有些人吃了优甲乐会兴奋，睡不着觉，所以一般都建议早上起床后再吃。出院那天，我微信问主管医生，饮食方面有什么要忌口的？至今没有回复，但从另外一个病友处得知，她的主管医生赵鹏医师告诉她，饮食方面没有避忌，烧烤都可以吃。当然，烧烤我是不敢狂吃的，按照网上的说法，全切后应该吃少油少盐的食物，避免过多食用含碘量高的食物，比如海带、海藻等。
综上所述，甲状腺全切后，日常生活中饮食方面的影响是：尽量吃少油少盐的食物，少吃烧烤油炸类的食物，不吃含碘高的食物；每天早上起床吃优甲乐2片，1个小时后才能吃早餐，4个小时内不能吃蛋白类的食物如豆类、奶类、蛋类等。除此之外，没有其他需要忌口的。当然，我看到网上也有很多病友不避忌这些，该吃什么就吃什么，现在也一样活得很好。个体不一样，患病程度不一样，每个人的生活习惯也不一样，不能一概而论。总之，这和其他癌症对比下，患甲状腺癌算是比较“幸福”的了。

下面通过时间来记录一下本人术后每个时段的生活状况。

#### 2024年9月12日
现在是手术后第15天，每天坚持吃优甲乐2片。葡萄糖酸钙片想起来就偶尔吃一两次，没有一直吃。甲钴胺片（修复神经）每天吃3次，每次1片。
身体方面没有明显的不适症，只是说话声音还嘶哑，但是相比刚手术出来那一周有一点改善了，至少能说得清楚一点了，虽然还是嘶哑。手术伤口还没完全好，感觉还有一点点流水。咽喉部吞咽有明显的异物感，可以明显感到咽喉部位不适。

听别的病友说，手术后十几天就会出大病理报告，也就是拿你手术割出来的甲状腺、肿瘤体、附近淋巴结等做全面的病理分析，判断癌细胞有没有转移，需不需要进一步治疗，比如做碘131放射治疗等。病友说，他当初是主管医生通过微信告诉他的。但直到现在，还没有任何人通过微信或电话告诉我术后病理报告，慢慢等吧。实在不行，1个月后复查的时候我再问医师。

#### 2024年9月13日
今天突然想到，老天让我患甲状腺癌，其实何尝不是给我一次再生的机会。通过患癌，让我知道我的身体情况有多糟，让我知道我以前的生活习惯和作息有多不健康。是的，我真的应该做出改变了。改掉生活暴饮暴食的习惯，改掉熬夜玩手机的习惯，改掉以前不爱运动的习惯，改掉以前爱发脾气的习惯。等我身体伤口恢复了，我就去跑步，坚持每天跑步2至3公里。听说，多做有氧运动可以抗癌，保持良好的心态也有助于抵抗疾病。

另外，我还在视频号看到有人说，经常喝一些中药也有助于抗癌，比如：白花蛇舌草(15-30g)+半枝莲(30g)+牛樟芝（红栓菌，8-10g），是肿瘤和艾症的克星，前两种各含100多种抗癌因子。我也是网上刷到的，没有验证过，大家不要轻易去试。前几天去咨询了这个视频号播主，她说像我的情况，可以预防复发，上面三种药材每周抓一小把煲15分钟，每周喝2次就好，每次隔1天；煲的时候最好放两三个红枣，去核；同时一定要注意忌口，不能大鱼大肉，多吃清淡、少盐少油、高营养的食物。（也有网上说，每隔两三周就煲来吃两三天就好了，不用一直吃。）如果没有确诊癌症的话，偶尔把白花蛇舌草+半枝莲当茶来喝，也可以散结节、消肿。

#### 2024年10月5日

甲状腺全切手术后一个月，9月30日下午17点左右在南宁市的广西医科大附属肿瘤医院做了复查，现在检查的报告出来了。肝、肾功能有一点指标不正常，打电话问了赵鹏医生，他说不要紧的。他说主要还是促甲状腺激素测定是5.079。每天吃了2片优甲乐，这个数值还有点高，赵鹏医生说再加大药量，要吃3颗优甲乐。一个月后再去他那里挂号做个复查看看。

#### 2024年10月10日

甲状腺全切手术后一个半月，伤口看起来已经愈合得差不多了。但是右胸口还是有时突然有点跳动，而且左转头时，脖子有很明显的拉扯感。说话声音还是有点沙哑，说话太长句会突然说不出来，卡住了，要吞一下口水停顿一下才能继续说话。喝水时，要慢慢喝，否则会呛到。

#### 2024年10月22日

手术快2个月了，说话声音终于不那么嘶哑了，差不多接近正常的声音了。但是还不能大声说话，大声的话依然会嘶哑。说明受损的神经也恢复了很多，所以我决定停止吃甲钴胺片，剩下的慢慢恢复吧。另外，钙片买了一瓶但是一直不怎么吃，只吃过几天吧，后面断断续续的，想起来就偶尔吃一下。现在声音恢复了一点，估计也不缺钙，决定还是停止吃钙片吧。吃太多药对身体不好。
另外，明显感觉记性越来越不好了，不知道是否跟手术有关。先观察一段时间，以后再说吧。
