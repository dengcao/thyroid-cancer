# thyroid-cancer

## 项目介绍

记录有关本人患甲状腺乳头状癌（Thyroid cancer）的一些情况。

## 关于本人

我叫邓振华，网名邓草，籍贯广西桂平，现住广西贵港。我是一名再普通不过的程序员，曾做过VB、ASP、HTML、CSS、Javascprit、PHP、Python等等程序语言的开发工作。我本科学历是计算机科学与技术专业毕业，工学学士学位，也曾参加自考获得中山大学行政管理本科专业文凭。2020年之前，曾在广东深圳、广州等一线城市的网络公司担任过技术主管、技术总监等职务，带过几十人的团队，开发过一些项目。2020年8月回老家，在贵港一家单位工作（合同工），虽工资不高，但也有五险一金，算是份养老的工作吧。虽然目前已确诊患甲状腺乳头状癌，不过大家也别担心程序项目更新的事，编程是本人最热爱的工作，只要我还活着，我就不会放弃这个工作。除此之外，本人还热爱文学，曾经写过小说。之所以公开患癌这个事，是为了告诉大家其实患癌也不是很可怕的一件事，既来之则勇敢面对，一起共勉吧！无论什么时候，也不要放弃。

# 以下记录本人患甲状腺癌的一些情况（持续更新）

## 2022年8月20日

我在贵港市东晖医院体检，首次查出甲状腺右叶囊性结节（TI-RADS：2级），桥本氏甲状腺炎。后来咨询医生得知，桥本氏甲状腺炎治疗不了，这个是自身免疫疾病，是自身免疫系统攻击甲状腺导致的。医生叫我定期观察即可。

## 2023年10月14日

在贵港市人民医院体检，查出右叶中段实性结节（TI-RADS：3类），大小1.2*0.7cm，边界清晰，低回声，内未见钙化，另：桥本氏甲状腺炎。与2022年体检时相比，说明这个结节增长了。挂号咨询了医生，依然叫我定期观察即可。

<img src="https://github.com/dengcao/thyroid-cancer/blob/main/images/2.2023%E5%B9%B4%E7%94%B2%E7%8A%B6%E8%85%BA%E4%BD%93%E6%A3%802_20231014.jpg?raw=true" width="70%" alt="2.2023年甲状腺体检2_20231014.jpg">

## 2024年8月8日

2024年8月8日本人到贵港市人民医院第二门诊体检，B超检查发现甲状腺结节至少3个：甲状腺右叶局灶性病变(一),TI-RADS 4a类；甲状腺左叶局灶性病变(一),TI-RADS 3类；甲状腺左叶局灶性病变(二),TI-RADS 2类。突然一下子检查这么多结节，而且还是TI-RADS 4a类，我也不知道是甲状腺结节一年不到的时间里突然长得这么快，还是之前就已经左右叶两边都长有了。第二天挂号了医生，建议我做活检。8月9日在贵港市人民医院做细针穿刺活检，病理分析结果：双侧甲状腺乳头状癌。

## 得知患癌一时懵逼住了，几天睡不好觉，也想不通为啥我就患癌了

得到检查结果的时候，感觉一下子晴天霹雳，有点接受不了现实，想不通，我每年都体检，为啥突然患癌了？而且我听说甲状腺癌是个懒癌，平时都是以年为单位的增长，为啥我的结节长得这么快？于是开始不断反思，可能是我近年来生活作息不规律，经常熬夜的原因；也可能是暴饮暴食，身体严重长胖的原因；朋友还说，是因为我长期献血，导致身体抵抗力下降……具体是什么原因，我不知道。问了医生，医生说献血一般不会导致抵抗力差，更不会导致患甲状腺癌，患甲状腺癌跟很多因素有关，比如遗传、生活习惯等等。遗传因素我感觉不太可能，家里的亲属没有患这个癌的，虽然我爸去年也查出甲状腺结节4a类，但穿刺检查是良性。而且身边的朋友和同事也很多患有甲状腺结节，感觉这个疾病更多的跟南方的气候和饮食习惯有关吧。但不管什么原因，我都得坦然勇敢面对这个现实。

## 2024年8月28日做了甲状腺全切手术

2024年8月21日，我到广西医科大学附属肿瘤医院（广西壮族自治区肿瘤医院）申请住院治疗，诊断：双侧甲状腺恶性肿瘤。住院做了各种检查，一周左右后，于8月28日做甲状腺全切手术。因为有同事在这家医院做过这个手术，有介绍到肿瘤医院耳鼻咽喉头颈外科副主任汪多平（主任医师，博士）比较厉害。于是我要求我的手术主刀医生是汪多平主任医师，虽然住院安排给我的主管医生是罗永标（主治医师），但听说就是汪多平给我做的手术。手术后，咽喉部位肿痛，吞咽困难，喝开水都被呛到。最明显的症状是声音嘶哑，至今（9月12日）没有恢复正常，医生说需要几个月恢复。术后，需要终身服用优甲乐补充甲状腺素，每天早上起床服用2片，1个小时后吃东西，4个小时内不能吃蛋白类食物，如大豆、牛奶、豆制品、蛋等等。还给我开了一个月的钙片（补钙）和甲钴胺片（修复神经）。一个月后再复查。

手术前，罗永标医生给我和家属做了术前谈话，主要内容是说手术有风险，需要家属和本人签字同意手术，手术可能会损伤喉返神经（说话嘶哑、喝水呛水等后遗症），可能会损伤甲状腺旁腺（缺钙等后遗症），如果是损伤了神经，一般需要3-6个月左右恢复，如果永久性损伤则终生无法恢复。住院期间，微信咨询了我的主管医生，但多数不回复，倒是手术后第三天查房时开始不停地催我赶快办理出院，还说换别人早就出院了，我也不知道为啥，我也没欠费，而且刚搬来新院区病床也不紧张。于是我厚着脸皮多住院了几天，因为我咽喉还肿痛，并且胸口的手术伤口还在隐痛和渗血水（每天从引流管流出大概30几毫升血水），手术后一周左右我才办理出院。可能催我出院也有他的原因吧，表示理解。此外，住院期间，所有医生、护士都对我很好。住院期间，汪多平主任来查房时曾跟我说，我的手术还是做得很成功的，叫我安心休养，缺钙和声音嘶哑的问题一般一个月左右就可以恢复正常。
